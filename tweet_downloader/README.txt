Downloads up to 3200 recent tweets from users.
  Does not include retweets.
  Run tweet_downloader.py
    "py tweet_downloader.py" will download tweets from the screen name specified in config.ini
    "py tweet_downloader.py <screen_name> will download tweets from the specified screen_name
    running run.bat will download tweets from the screen names listed in the file.
    config requires token and keys
